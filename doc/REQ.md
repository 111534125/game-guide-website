1. 專題名稱 / Project Title / 프로젝트명

遊戲攻略與社群網站 (Game Strategy & Community Hub)

2. 專案概述 / Project Summary / 프로젝트 개요

💡 專案目的 / Purpose / 목적

建立一個高品質、即時更新的遊戲攻略網站，讓玩家能夠：

快速查詢最新的遊戲新聞、更新情報與詳盡的攻略指南。

獲取遊戲技巧、角色建構（Builds）或任務流程等實用資訊。

建立一個玩家社群，交流遊戲心得與提問。


🎯 主要使用者 / Target Users / 주요 사용자

角色

核心需求

玩家 (Gamers)

瀏覽遊戲攻略、查詢資訊、閱讀新聞、在社群區發言。

內容編輯/貢獻者 (Editors/Contributors)

撰寫並發布攻略文章、更新遊戲情報。

4. 成功指標 / Success Metrics / 성공 지표

這些指標將用於衡量網站是否達到專案目的：

跨裝置相容性： 網站能正常且美觀地顯示於電腦與手機瀏覽器（RWD 響應式設計）。

內容更新頻率： 每週至少更新 5 篇新的攻略文章或遊戲新聞。

使用者參與度： 攻略文章的平均留言或互動數達到設定基準（例如：每篇文章至少 5 則評論）。

5. 非範圍內容 / Out of Scope / 범위 밖

以下是專案階段初期不會包含在內的內容：

使用者註冊與登入系統： 不包含複雜的會員登入、權限管理與密碼重設等功能。所有內容發布與管理透過後台（Out of Scope）進行，使用者評論功能將使用匿名或第三方嵌入式服務（若無登入需求）。

金流功能： 不含任何付費內容、訂閱或捐款功能。

獨立手機應用程式 (App)： 不包含 iOS 或 Android 上的原生手機 App。
